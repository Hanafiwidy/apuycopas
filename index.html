    Tarik Data dari Server

    Remote via PuTTY ke IP 10.0.3.3, masuk ke cd satkom/svr/APR, jalankan script AWK.

    Paste ke PuTTY via klik kanan setelah copy
  • 2

    Pilih Tab & Paste Data

    Pilih tab di sidebar — Traffic SMS Real Time untuk raw log PuTTY, Traffic WhatsApp untuk WABA, atau Traffic SMS untuk tabel report harian.

  • 3

    Process & Export ke Excel

    Klik Process Data, lalu gunakan tombol Copy per tanggal atau Copy Full Tabel untuk paste langsung ke Excel.
