Selamat Datang di Apuy Copas Copas

Automasi paling gg. Parsing data log Trafik SMS MDM & JATIS menjadi format tabel ganteng, siap untuk di-export ke Excel hanya dalam hitungan detik anjay.

Cara Penggunaan Tools

  • 1

    Tarik Data dari Server

    Lakukan remote menggunakan PuTTY ke IP 10.0.3.3 dan masuk ke cd satkom/svr/APR , masukan command "klik copy command putty" ambil raw data trafik.
    💡 Tip: Copy script awk di bawah ini lalu paste diputty!

  • 2

    Copy & Paste

    Buka menu Dashboard Analytics di sidebar kiri, lalu paste seluruh teks log (AGGR : MDM / JTS) ke dalam kolom yang disediakan.

  • 3

    Process Data

    Klik tombol Process Data. Sistem akan mengkalkulasi total per jam, peak hour, dan provider tertinggi secara otomatis.

  • 4

    Export ke Excel

    Gunakan tombol Copy Full Tabel untuk menyalin kerangka utuh, atau Copy Nilai Provider Saja jika ingin mem-paste angkanya saja ke template laporan Excel Anda.
