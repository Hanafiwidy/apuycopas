Panduan Penggunaan
  • 1

    Tarik Data dari Server

    Remote via PuTTY ke IP 10.0.3.3, masuk ke direktori cd satkom/svr/APR, lalu jalankan script AWK di bawah.

    Paste ke PuTTY via klik kanan setelah copy
  • 2

    Pilih Tab & Paste Data

    Pilih tab yang sesuai di sidebar — Dashboard Analytics untuk raw log PuTTY, Traffic WhatsApp untuk data WABA, atau Traffic SMS untuk tabel report SMS.

  • 3

    Process & Export ke Excel

    Klik Process Data, lalu gunakan tombol Copy Tabel atau Copy Nilai Saja untuk langsung paste ke file laporan Excel.
